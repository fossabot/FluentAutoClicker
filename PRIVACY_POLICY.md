# Privacy Policy

**Effective Date:** January 15, 2025

## 1. Introduction

This Privacy Policy describes how Fluent Auto Clicker ("we," "our," or "the Application") collects, uses, and protects your information when you use our Windows desktop application.

By using the Application, you consent to the data practices described in this policy.

## 2. Information We Collect

### 2.1 Local Application Data
The Application stores the following information locally on your device:
- User preferences and settings (click intervals, mouse button preferences, etc.)
- Application configuration data
- Theme and UI preferences

### 2.2 No Personal Information Collection
We do **not** collect, store, or transmit:
- Personal identifying information
- Usage patterns or behavior data
- Mouse click locations or targets
- Screen content or screenshots
- Network activity or browsing history

### 2.3 System Information
The Application may access standard Windows system information for functionality:
- Mouse and keyboard input APIs
- System power management settings
- Display and window management information

This information is used solely for application functionality and is not collected or transmitted.

## 3. How We Use Information

### 3.1 Local Data Usage
Information stored locally is used to:
- Maintain your application preferences between sessions
- Provide the auto-clicking functionality you configure
- Remember your UI theme and layout preferences

### 3.2 No Data Transmission
The Application operates entirely offline and does not:
- Send data to external servers
- Connect to the internet for operation
- Share information with third parties
- Upload or sync data to cloud services

## 4. Data Storage and Security

### 4.1 Local Storage
All application data is stored locally on your device using Windows standard application data folders. This data is:
- Encrypted using Windows built-in security features
- Accessible only to your user account
- Protected by your Windows login credentials

### 4.2 No External Storage
We do not store any of your data on external servers or cloud services.

### 4.3 Data Retention
Your settings and preferences are retained until you:
- Uninstall the Application
- Manually delete the application data
- Reset the Application settings

## 5. Third-Party Services

### 5.1 Microsoft Services
The Application uses Microsoft Windows APIs and services:
- **Windows Input APIs**: For sending mouse click events
- **Windows App SDK**: For application framework
- **Microsoft Store**: If distributed through the store (usage analytics may apply)

### 5.2 Microsoft Privacy Policy
When using Microsoft services, Microsoft's privacy policy applies:
- [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement)
- Microsoft may collect telemetry data about application usage if distributed through the Microsoft Store

### 5.3 Windows Telemetry
Windows may collect diagnostic and usage data about installed applications according to your Windows privacy settings. This is controlled by:
- Windows Privacy Settings
- Microsoft's data collection policies
- Your consent during Windows setup

## 6. Third-Party Links

### 6.1 External Links

### 6.2 External Privacy Policies

## 7. Children's Privacy

The Application does not knowingly collect information from children under 13. The Application is not specifically designed for use by children, and we recommend parental supervision for any automated clicking tools.

You can control Windows telemetry and diagnostic data through:
